📖 Explicação Técnica do Projeto

🏷️ HTML Semântico

Utilizamos tags como <nav>, <main>, <section> e <article> para que o navegador e ferramentas de acessibilidade entendam a hierarquia e o significado de cada parte do conteúdo, indo além de simples <div>s[cite: 11, 19].

🎨 CSS: Layout e Design

  • Box Model: Cada elemento é tratado como uma caixa com margens, bordas, preenchimento (padding) e o conteúdo em si, permitindo controle total do espaçamento[cite: 36, 38].
  • Flexbox: Usado no menu de navegação e na seção de personagens para alinhar elementos de forma flexível e centralizada[cite: 19, 36].
  • Grid: Aplicado na lista de agentes para organizar os dados em colunas estruturadas de forma automática[cite: 26, 36].
  • Responsividade: Através de Media Queries, o site se adapta a celulares, reorganizando as colunas do Grid em uma linha única[cite: 23, 26].

⚙️ JavaScript e Persistência

  • LocalStorage: Os dados dos agentes são salvos diretamente no navegador. Isso garante que as informações não sumam quando você fecha ou atualiza a página[cite: 17, 36].
  • Fetch API: Na tela de Localização, o JavaScript faz uma "chamada" externa para o serviço ViaCEP, buscando o endereço em tempo real sem precisar recarregar o site[cite: 11, 29, 36].
